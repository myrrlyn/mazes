# Mazes

This is my work from reading the book [*Mazes for Programmers*][book].

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mazes'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mazes
